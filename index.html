数当てゲーム

1~10の乱数を生成してその乱数の数値と同じ数値であったときに正解とし間違いであれば正解の数値からどれくらいなのかを判断して言ってくれるゲームを作成した。また、そのゲームは4回動作するように作成したがゲーム自体は3回で終わるように作成した。最初は4で固定するように作った。

そしてこれに、label要素,input要素,button要素を加えてボタンをつけてボタンを押すというイベントを利用して乱数を発生させて、その後に、同じ値かどうかの判断を行うゲームになるように改良した。動作方法は皮えらず。4回動作するようにして3回でゲーム自体は終わるようにして作った。

各授業回の学習事項

第1回 WWW と HTML

HTMLの基本的な文書の構成、要素の種類、見出しや小見出し、リンクの属性の作り方、コメントの書き方や画像の埋め込みについて学んだ。また、HTMLで改行してもパージでは改行されないので改行のやり方についても学んだ。

演習の成果： 演習1-4HTMLのさまざまな要素

第2回 CSS によるWeb ページの作成の装飾

CSSを使ってwebページのレイアウトやフォント、色などを変更する方法について学んだ。また、タイトルにグラデーションを付けるなど背景を変える方法についても学んだ。さらに、箱の作り方やアイコンの位置を変える設定についても学んだ。

演習の成果： 演習2-3: ウェブブラウザ説明ページの装飾

第3回 Java Script の基本

JavaScriptを用いたコンソール上でのプログラムの書き方について学んだ。そして、webサイトで開発ツールを用いたデバックのやり方についても学んだ。また、for-of構文を使ったオブジェクトの参照方法についても学んだ。さらに、javaとjavaScriptの違いについても学んだ。

演習の成果： 演習3-6: オブジェクトの参照

第4回 DOM と Java Script

ボタンを押したらページが変化するという処理をJavaScriptプログラムを用いて記述する方法について学んだ。その中で、フォーム要素、イベント処理、DOMの役割と書き方について学んだ。また、それらを実現するためにinput要素やtype属性の種類などについて学んだ。

演習の成果： 演習4-10: 足し算ページ
